# 📌 iTask – Todo Manager

A minor React project for managing daily todos with a clean UI and persistent storage.

---

## ✨ Features

- ✅ Add new todos
- ✏️ Edit existing todos
- 🗑️ Delete todos
- ☑️ Mark todos as completed
- 👁️ Show / Hide completed todos
- 💾 Persist data using **LocalStorage**

---

## 🛠 Tech Stack

| Technology | Details |
|---|---|
| **React** | Vite / Create React App |
| **JavaScript** | ES6+ |
| **CSS** | Tailwind CSS |
| **Icons** | react-icons |

---

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or above recommended)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/itask.git

# Navigate to the project directory
cd itask

# Install dependencies
npm install

# Install react-icons
npm install react-icons --save

# Start the development server
npm run dev
```
